亞馬遜廣告投放軟體

專為亞馬遜賣家打造的一站式廣告管理平台。系統以 AI 智慧競價與自動優化功能為核心，全面提升廣告效益與操作效率。

專案概覽


目標用戶

  • 亞馬遜 跨站營運賣家
  • 廣告代理商與品牌商家
  • 多店鋪電商營運團隊


競業分析

競品工具介面繁雜無 AI 優化，操作效率低。 (現有 Amazon 廣告管理工具大多介面複雜、缺乏智慧優化功能，導致賣家操作繁瑣效率低下。)


設計目標

設計目標在於以使用者為中心打造廣告管理解決方案。透過完善的 SaaS 系統流程、清晰的視覺層級與模組化功能佈局，提高用戶廣告操作效率並增強數據分析決策能力。

User Interface Design

Dashboard

Performance Overview

以Performance Card 直觀呈現廣告整體表現概況。預設顯示 Total Sales、Ad Sales、Ad Spend 以及 ACOS 三大核心指標，並以色彩和百分比標示與前期相比的趨勢變化。使用者也可自訂關注的指標組合，確保儀表板資訊符合其業務目標。

Performance Over Time

提供廣告績效的時間趨勢視覺化分析。預設提供兩組關鍵趨勢圖：一組對比 Ad Sales 與 Ad Spend，另一組對比 Ad Orders 與 ACOS，分別以折線圖或長條圖呈現相關走勢。使用者可以自由選擇與調整圖表中的數據指標，深入觀察廣告成效的歷史變化，迅速識別趨勢與潛在問題。

Preview Report

將儀表板資料匯整為報告格式，方便下載與分享。使用者可在預覽頁檢視所選的關鍵績效指標與趨勢圖表，並視需要移除不必要的內容。最終可快速匯出專業的廣告績效報告，方便用於內部檢視或向客戶報告廣告成果。

User Interface Design

Ad manager

AI Bidding

AI Bidding（AI 智慧競價）透過每日自動分析廣告表現並套用使用者設定的 Target ACOS（目標廣告成本佔比），自動計算並調整競價。此功能根據廣告效益目標動態優化出價，減少人工調整負擔，確保廣告投放維持在理想 ACOS 範圍內。

Auto-Harvesting

Auto-Harvesting（自動搜尋詞收割）會每日分析搜尋詞報告，自動篩選高效關鍵字投入廣告，並透過 Keyword Isolation（關鍵字隔離）排除低效或無關鍵詞。系統將高相關搜尋字詞自動加入投放清單，並將低效詞設為否定關鍵字，不斷優化廣告目標對象，同時減少人工篩選的負擔。

Filter

提供多維度篩選條件，支援依據 Marketplace、Campaign Type、狀態（Active/Paused）、ACOS 區間等進行快速資料篩選。篩選條件可即時套用至 DataGrid 呈現結果，使用者能迅速聚焦在目標廣告活動上，無需翻找冗長報表。此設計有效提升操作效率並支援日常管理與例行優化任務。

Filter

為解決 Amazon 原生六層級廣告架構操作不便問題，Ad Manager 將 Campaign、Ad Group、Keyword、Target、Product Target、Search Term 等六大資料階層整合為可視化頁籤式切換。使用者可在同一頁面中橫向切換不同階層資料，不僅保留層級邏輯，也讓分析流程更加順暢且直覺。每個頁籤下的欄位皆依資料類型設計，並支援排序與欄位自定義，大幅簡化使用者的操作成本。

User Interface Design

Campaign Builder

Campaign Builder 為賣家提供結構化且模組化的廣告活動建立流程。使用者可依據不同行銷目標選擇策略模板，搭配多層級商品挑選與自動競價設定，大幅縮短建置流程時間，同時確保每個 Campaign 具備最佳結構與資源配置。整體介面流程以引導式動線設計為主，適用於初階與進階使用者，實現可複製、可標準化的廣告投放流程。

Goal Strategy

提供 Basic、Advanced、Brand-Based、Custom 四種廣告策略模板，滿足不同投放目標需求。介面以圖解方式即時預覽各模板將建立的 Campaign 與 Ad Group 結構，幫助使用者清晰理解複雜的廣告佈局。Basic/Advanced/Brand-Based 三種策略分別側重於基礎優化、高階競價及品牌戰略投放，而 Custom 則允許有經驗的使用者自訂單一 Campaign 結構。模組化的策略設計確保使用者找到最契合自身需求的方案，快速完成廣告活動建立。

Products

用戶快速選擇欲投放廣告的商品。支援以 ASIN、SKU 或產品名稱搜尋商品，並提供 Child View 與 Parent View 兩種視圖模式切換。Child View 僅顯示單一子 ASIN 結果；切換為 Parent View 則可檢視父子變體的完整結構，方便逐一挑選多樣化產品。所有選取的商品會列入右側清單集中管理，用戶可個別移除，或一鍵清空全部選擇。此外，系統提供 Competing Campaigns 提示，顯示所選商品是否已有廣告活動在運行，以避免重複建立新的廣告活動。
