丹山E學霸線上學習App

創新的線上英語學習平台，讓孩子透過與美國頂尖學生的互動，提升英語能力並拓展思維視野。平台提供豐富的影片學習資源，讓學生能隨時隨地學習。經驗豐富的導師全程引導啟發，培養主動學習態度，幫助孩子全面提升英語技能。

專案介紹
UX Research

使用者體驗研究

STEP - 01
目標用戶
Danshan EScholar 的核心用戶是 介於 12 至 20 歲 的青少年設計，特別是希望提升英文口說能力、強化跨文化交流、並習慣自主學習的學生族群。透過互動式課程與優質導師資源，提供靈活且具啟發性的學習環境。
STEP - 02
競業分析
市面常見兒童英語平台如 VIPABC Kids、Cambly Kids、Duolingo，強調外師、遊戲化或彈性學習。我們整合其優勢，透過「學霸」導師與影音資源打造差異化體驗。
STEP - 03
設計目標
我們的目標是設計出一個溫暖、直覺、好上手的兒童英語平台，讓學生、導師與家長皆能輕鬆使用。版面與互動皆以使用者需求為核心，提升學習動機與互動頻率。

Design Guideline

產品設計規範

字體與配色

採用 Roboto 無襯線字體，大標題 600–700、內文 400，行高與字距微調增進閱讀舒適度。配色以黃色 為主，輔以藍色強調，並用三種灰階作為背景和次要文字，兼顧品牌調性與清晰度。

元件規格

所有元件皆採卡片化風格：8px 圓角、16px 基本間距。按鈕分實底與細框兩種樣式，含 Hover/Disabled 狀態；表格與輸入框維持 16px 內邊距與細線框；Icon 採 20×20px 線性風格，主色與灰階提示互動狀態。

User Interface Design

課程管理

學員

提提供完整的功能模組，包含預約課程、線上學習、教學成果評鑑等機制，讓學生能夠根據個人進度自由安排課程。平台同時設有學習歷程追蹤與提醒系統，協助學生自我檢視並持續優化學習策略。

學霸

學霸導師可於平台上備課、管理教案，並建立個人教學平台頁面，展示其教學理念、專業領域與學生回饋。此系統也有助於導師累積教學履歷，提升其在平台的信任度與能見度。

User Interface Design

會員中心管理

完整的一站式會員中心，整合個人資訊、學習歷程、導師互動、課程通知與即時反饋機制，為學生與導師提供流暢、個別化的操作體驗。 學生可於平台中編輯個人資料、查看每堂課的進度與評分、接收即時上課通知與課程更新提醒，並能自由追蹤學霸導師、管理訂單與繳費紀錄。同時提供反饋與疑難排解通道，讓學習歷程更透明、教學溝通更即時有效，確保雙方能專注於教與學的核心。

User Interface Design

先聽再學，成效可視

提供免費線上試聽與教學評鑑服務，讓學生在報名前先了解課程風格，課後則可取得個別化的學習回饋與成效追蹤報告，學習更有方向。

線上試聽服務

平台提供線上試聽機會，讓新用戶可在正式訂閱前親自體驗課程內容、互動模式與教學品質。此設計可大幅提升使用者初期的信任感與參與意願，同時也作為導師個人品牌的展示窗口。

線上評鑑平台

平台內建學習回饋與教學評估系統，學生於每堂課後可即時收到導師回饋報告，了解自己的優勢與待加強項目。導師亦能根據學生表現，量身規劃後續教學策略，強化學習成效並提升課程精準度。
